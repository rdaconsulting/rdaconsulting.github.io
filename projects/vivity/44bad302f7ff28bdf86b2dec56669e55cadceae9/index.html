Final primary outcome model with mesopic pupil size and interaction

Because the Mesopic Pupil Size was significantly associated with BCDVA, I made a final model (below) which included the two IOL tilt/decentration exposures, as well as mesopic pupil size, and an interaction term for each. In this final model, decentration is marginally significant (p=0.0237), and the interaction term also appears to be (nearly) marginally significant (p=0.0575); thus, it’s possible mesopic pupil size modifies the relationshi pbetween decentration and BCDVA. Given all the tests we’re doing, this could be simply a spurious finding, but I thought I’d point it out in case there might be a biologically plausible reason for this finding.
